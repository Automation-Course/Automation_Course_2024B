
how to use the calculator:

run the code
there will be a pop up window that will as you which convert you want to do:
binary to decimal or otherwise?
then another pop up window will pop, enter the number you want to convert

attention!!: if you choose binary to decimal please enter a binary number or the calculator will throw you an exception

after you enter your number, another pop up window will show with your answer.
after that you can enter another number as you wish
