על מנת להשתמש במחשבון:
הריצו את הקוד.
לאחר מכן מסך בחירה יקפוץ למשתמש המאפשר לבחור: המרה מבינארי לעשרוני או ההפך.
לאחר שנבחר באפשרות הרצויה, יקפוץ חלון נוסף להכניס מספר מתאים בחלונית בהתאם לאופציה שהמשתמש בחר.
נכניס את המספר ונלחץ GO.
הקוד ירוץ ברקע ויחשב את ההמרה של המספר ויציג את התוצאה למשתמש בחלונית חדשה.
לאחר מכן המשתמש יוכל ללחוץ OK ולהמיר מספר נוסף.
