הוראות למפעיל – מחשבון המרה בין בסיסים:
1. הפעל את התוכנית.
2. בחר מהתפריט את האופציה המציעה את הבסיס שתרצה להמיר, והזן 1\2 לפי בחירתך.
3. כתוב את המספר אותו תרצה להמיר בהתאם לבחירתך ולחץ ENTER.
4. לאחר הלחיצה יופיע לך במסך המספר המומר לפי הזנתך.
5. אם תרצה לבצע המרה נוספת יש להזין "Yes", אחרת "No".
6. כל הזנה של קלט אחר תסיים את ריצת התוכנית.
