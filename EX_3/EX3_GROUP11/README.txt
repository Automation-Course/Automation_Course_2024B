HOW TO USE THE CHIP:


1. ENTER THE USB TO THE COMPUTER.
2. ENTER THE ARDUINO IDE.
3. COVER THE PHOTORESISTOR(DONT LET THE LIGHT IN, SO IT WILL THINK ITS DARK). 
4. RUN THE CODE.
5. GIVE THE PHOTORESISTOR LIGHT - SERVO WILL SPIN, AND THE MUSIC WILL START PLAYING
ALSO THE LCD WILL PRINT A MESSAGE "HAVE A GOOD LEARNING DAY"
THE LIGHT BULB WILL TURN ON.
6. START AGAIN? - COVER THE PHOTORSISTOR AGAIN AND THEN UNCOVER IT.
